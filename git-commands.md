# Git Commands I Learned and Applied in This Repo

git init - starts tracking a folder as a git repo

git config --global user.name "name" - sets the name shown on my commits

git config --global user.email "email" - sets the email shown on my commits

git status - shows what changed, what's staged or not

git add . - stages all changed files, ready for commit

git commit -m "message" - saves a checkpoint with a message

git push - sends my commits to GitHub

git pull - brings changes from GitHub into my local repo

git log - shows my commit history

git remote add origin <url> - connects my local repo to a GitHub repo

git remote -v - shows which GitHub URL my repo is connected to

git branch -M main - renames my branch to main

git push -u origin main - first push, links local branch to GitHub

Move-Item -Path "source" -Destination "destination" - moves a file or folder (PowerShell command)

Remove-Item -Path "folder" -Force - deletes a folder (PowerShell command)
